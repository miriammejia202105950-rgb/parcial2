Proyecto de Software

Un proyecto de software es el conjunto de actividades necesarias para desarrollar un producto informático que cumpla ciertos requisitos y brinde una solución a un problema o necesidad.

Tipos de proyectos de software

  • Aplicaciones web
  • Aplicaciones móviles
  • Aplicaciones de escritorio
  • Sistemas embebidos
  • Desarrollo
