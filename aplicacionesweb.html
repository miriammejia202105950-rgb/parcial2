Aplicaciones Web

Frontend

El frontend es la parte visible de una aplicación web, donde interactúa el usuario.

  • HTML: estructura del contenido.
  • CSS: diseño y estilos visuales.
  • JavaScript: interactividad y dinamismo.

Backend

El backend es la parte lógica y funcional que procesa los datos del sistema.

  • Bases de datos: MySQL, MongoDB, PostgreSQL.
  • Servidores web: Apache, Nginx.
  • Lenguajes: PHP, Python, Java, Node.js.

Full Stack

Un desarrollador Full Stack domina tanto el frontend como el backend, permitiéndole crear aplicaciones completas.
